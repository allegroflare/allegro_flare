AllegroFlare::TileMaps::TileMesh Methods

This class is the evolutionary replacement for TileMaps/PrimMesh. It utilizes vertex_buffer and includes features for postponing vertex_buffer refreshes for better performance.

initialize 
void initialize()

Initializes the mesh, sets the internal initialized state to true, and performs the initial resize based on the current column and row counts.

Guards:
  • Object must not already be initialized.
  • Allegro system must be installed.
  • Allegro primitives addon must be initialized.
set_num_rows 
void set_num_rows(int num_rows = 0)

Sets the number of rows in the tile mesh. If the mesh is already initialized, it triggers a resize.

set_num_columns 
void set_num_columns(int num_columns = 0)

Sets the number of columns in the tile mesh. If the mesh is already initialized, it triggers a resize.

set_tile_ids 
void set_tile_ids(std::vector<int> tile_ids = {})

Batch sets the tile IDs for the entire mesh. Note: This method currently does not account for flipped tiles. It marks the vertex buffer as dirty if updates are being held, otherwise it refreshes it immediately.

remove_tile_xy_from_index 
void remove_tile_xy_from_index(int tile_x = 0, int tile_y = 0)

Marks a specific tile to be removed from the rendering index. Effectively "hides" the tile from being drawn.

Internal helper that collects all vertex indices associated with tiles that have been marked as "removed".

void refresh_index_vertices_from_removed_tiles_and_refresh_index_buffer()

Rebuilds the index vertices by excluding those belonging to "removed" tiles, then updates the hardware index buffer.

int remove_vertices_from_index_vertices(std::vector<int> vertices_to_remove = {})

Removes specific vertices from the index list and triggers an index buffer refresh.

std::vector<int> vertex_indices_for_tile_xy(int tile_x = 0, int tile_y = 0)

destroy 
void destroy()

Properly destroys the Allegro vertex and index buffers and nullifies their pointers.

disable_holding_vertex_buffer_update_until_refresh 
void disable_holding_vertex_buffer_update_until_refresh()

Sets the mesh to update the vertex buffer immediately whenever a tile changes, rather than waiting for an explicit refresh.

get_vertex_buffer_size 
std::size_t get_vertex_buffer_size()

Returns the number of vertices currently stored in the hardware vertex buffer.

get_index_buffer_size 
std::size_t get_index_buffer_size()

Returns the number of indices currently stored in the hardware index buffer.

resize 
void resize(int num_columns = 0, int num_rows = 0)

Resizes the mesh to the specified dimensions, clearing all current tile data and recreating the vertex/index buffers.

render 
void render(bool draw_outline = false)

Draws the tile mesh using indexed buffer rendering. Can optionally draw a magenta outline around the entire mesh boundary.

Guards:
  • Mesh must be initialized.
  • Atlas must be set.
  • Buffers must not be dirty.
set_tile_id 
bool set_tile_id(int tile_x, int tile_y, int tile_id, bool flip_h = false, bool flip_v = false, bool flip_d = false, ALLEGRO_COLOR color = WHITE)

Sets the ID and transformation (flipping) for a specific tile at (x, y). Returns true if successful. Handles UV coordinate calculation and flipping logic.

int get_tile_id(int tile_x = 0, int tile_y = 0)
